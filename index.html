What's inside

Everything a fintech needs.

Real patterns, not toy implementations. Each module is built the way it would be in production.

🏦

Crypto Wallets

Multi-currency wallets with live balances. Each wallet has its own URL, its own ledger, and a full transaction history.

📊

Live System Map

Real-time graph of all entities and their relationships via Vue Flow. Watch the system topology update as you interact.

✅

Compliance & KYC

Risk scoring, KYC status tracking, and AML checks. Each user carries a compliance state machine that drives access rules.

📒

Double-Entry Ledger

Every money movement is a proper ledger entry — debit and credit in one atomic operation. No silent balance mutations.

🎨

Design System

Token-based components in packages/ui. Zero invented values, zero one-off divs, zero magic numbers.

🤖

AI-Native Workflow

Agents follow strict guides, a clean-code gate, and a three-role system — analyst, builder, reviewer. Below 99% compliance? It doesn't ship.
