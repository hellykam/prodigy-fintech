# Research

This folder collects research that informs product decisions.

## Structure

- `legacy-figma/` — features extracted from the existing Figma (done in separate chat)
- `competitors/` — competitor analysis (Coinify, MoonPay, Ramp, Transak)
- `kyc-risk/` — KYC/AML best practices, risk scoring, FATF Travel Rule
- `widget-patterns/` — embed code patterns, customization APIs, white-label approaches
- `admin-patterns/` — fintech admin UX patterns, dashboards, risk queues
